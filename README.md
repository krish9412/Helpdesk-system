# Helpdesk Ticketing System 

A full-stack Flask-based IT Helpdesk system.

## Features
- Incident ID system (INC0001 format)
- Ticket categories (Hardware, Software, Network, etc.)
- Role-based login (Admin, Technician, Employee)
- Ticket assignment system
- Status tracking (Open, In Progress, Resolved, Closed)
- Dashboard with live stats

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite
- Bootstrap
